# Jinja reference for dbt

Core delimiters:
- `{{ ... }}` expressions
- `{% ... %}` statements
- `{# ... #}` comments

Most-used dbt-specific helpers:
- `ref()`
- `source()`
- `env_var()`
- `var()`
- `target`
- `this`
- `run_query()`
- `log()`
- `adapter.dispatch()`

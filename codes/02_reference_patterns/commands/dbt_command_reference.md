# dbt command reference

Grouped by workflow:

- bootstrap: `dbt init`, `dbt debug`, `dbt deps`
- graph inspection: `dbt ls`, `dbt parse`, `dbt compile`, `dbt show`
- build path: `dbt seed`, `dbt run`, `dbt test`, `dbt build`, `dbt snapshot`
- docs: `dbt docs generate`, `dbt docs serve`
- source quality: `dbt source freshness`
- state and recovery: `dbt clone`, `dbt retry`
- operations: `dbt run-operation`, `dbt clean`

Use this file alongside Appendix T in the book.

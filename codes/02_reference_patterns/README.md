# 02_reference_patterns

이 폴더는 companion pack의 “고급 참고 자료”입니다.

여기에 들어 있는 예시는 아래와 같습니다.
- governance / contracts / versions / grants
- semantic layer / metrics / saved queries
- functions / UDF
- mesh / dependencies.yml
- slim CI / state / defer / clone
- platform profiles
- microbatch / materialized_view / dynamic table 메모

주의
- 일부 기능은 dbt 버전/엔진/플랜/어댑터에 따라 문법과 지원 범위가 달라질 수 있습니다.
- 이 폴더는 “설계 참고용”으로 보고, 실제 적용 전에는 공식 문서와 현재 사용 중인 runtime을 다시 확인하세요.
